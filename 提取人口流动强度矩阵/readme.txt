GetRepeatedID.py
该文件用于提取在某地重复出现的用户id。输入为原始手机通信数据，输出为只保留重复出现用户的手机通信数据。该操作的意义是保留表现出‘停留’行为的用户，去掉表现出‘过路’行为的数据。

integrate.py
该文件用于将GetRepeatedID.py文件的输出整合成一个文件。

GetHospital2ID_2.py
该文件用于将处理过的数据与每个医院附近基站进行匹配，最终输出的文件每一行为在某医院附近停留的用户的id，保存为文件，假设文件名为Hospital2ID_2.csv

GetLU2_ID_2.py
该文件用于将处理过的数据与每个居住区附近基站进行匹配，最终输出的文件每一行为在某居住区附近停留的用户的id，保存为文件，假设文件名为LU2ID_2.csv。

GetR.py
该文件用于计算人口流动矩阵R。文件输入为Hospital2ID_2.csv、LU2ID_2.csv以及医院和居住区的基站坐标文件。输出为R矩阵。
